Problem 2 宣告函式計算表面積

sorce code

double getSurfaceArea(double _radius, double _height)
{
    const double pi = 3.14159;
    return 2*pi*_radius*_height + 2*pow(_radius, 2)*pi;
}

main()與終端機輸出 善用函式可以大幅增進程式的可讀性，把許多與整體應用程式邏輯脈絡無關的內容移至main()函式之外。一個函式應該只負責一項很鮮明的工作，不應該試圖讓同一函式身兼多職。把不同功能寫在不同函式裡，讓函式看起來意義更明顯，也更方便命名。
